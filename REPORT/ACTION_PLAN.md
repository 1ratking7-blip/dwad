### Почему не "30 постов + 30 твитов + 20 статей + 20 Quora-ответов" по чек-листу
Бриф просил конкретные объёмы контента по каждому каналу. Такое количество имеет смысл только
если под каждый канал есть реальный, проверенный сценарий размещения (нашёл 5-7 подходящих
площадок под конкретный формат) — штамповать 20 "статей" под площадки, которые категорически
не подходят (Medium, Pinterest, AlternativeTo), было бы объёмом ради объёма, а не шагом к
доходу. Контент выше покрывает все каналы, которые реально прошли фильтр по нише/политикам
площадок; расширение объёма по каждому — по запросу, когда конкретный канал начнёт давать
трафик и понадобится больше материала.
