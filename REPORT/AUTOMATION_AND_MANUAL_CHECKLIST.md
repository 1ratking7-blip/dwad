# Автоматизация: что реально можно, и чек-лист ручных действий

## Что можно автоматизировать безопасно

| Что | Как | Почему безопасно |
|---|---|---|
| Публикация в собственный Telegram-канал | Встроенный "Schedule message" в Telegram, либо Telegram Bot API + `Scripts/content_generator.js` (уже генерирует контент) | Ваш собственный канал — не чужая площадка, нет риска ToS-бана за спам |
| Discord-сервер (`!spin`/`!bonus`/`!faq`/`!leaderboard`) | Уже реализовано — `Scripts/discord-bot/` | Ваш собственный сервер |
| Генерация черновиков контента (Telegram/Twitter/email) | `Scripts/content_generator.js` уже это делает | Генерация — не публикация; черновик всё равно проверяется вами перед постом |
| Мониторинг упоминаний бренда ("ZHELEZO", "BC.Game" + ваш домен) | Google Alerts / Яндекс.Алерты — бесплатно, email-уведомления | Официальный, легальный сервис мониторинга |
| Мониторинг позиций сайта в поиске | После подключения GSC/Яндекс.Вебмастер — встроенные отчёты по позициям, бесплатно | Официальные инструменты поисковых систем |

## Что НЕ стоит автоматизировать (риск бана/спама)

| Что | Почему |
|---|---|
| Автопостинг на bitcointalk/форумы | Модерация форумов явно банит паттерны автоматического постинга; ценность там именно в живом, контекстном участии |
| Автоответы на Quora | Тот же риск — Quora определяет "little value, mainly directing traffic" именно по шаблонности |
| Любой автопостинг рекламы казино в VK/группы | VK категорически запрещает казино-контент (см. исправление в `GROWTH_PLAN.md`) — автоматизация здесь не спасёт, сам контент запрещён |
| X/Twitter — полностью автоматический бот-постинг без проверки | Технически возможно через легитимные шедулеры (Buffer, TweetDeck/X Pro, Hypefury — работают через официальный API, не банятся сами по себе), но контент всё равно стоит проверять вручную перед публикацией, особенно органические твиты — шаблонность видна аудитории |
| Покупные backlink-сервисы | Уже отклонено в `casino_directories.md` — grey-hat SEO, риск пессимизации домена в YMYL-нише |

## Рекомендуемые инструменты для легитимного шедулинга (X/Twitter)
- **TweetDeck / X Pro** — официальный инструмент X, бесплатное планирование постов.
- **Buffer** (free tier достаточно для 1 аккаунта, ограниченное число постов/месяц).
- Оба работают через официальный API X — не создают риска бана за "автоматизацию", в отличие
  от скриптов, эмулирующих браузер/логин.
